## Common Lookup Queries

### For Integration/Coding:
- **What fields are in Section X?** → Read that section's line range
- **How is [score] calculated?** → Usually in section or CAP appendix
- **What are the valid codes for [field]?** → Look in that field's "Coding" subsection

### For Clinical Understanding:
- **Assessment workflow** → Section B (Intake)
- **Cognitive assessment** → Section C
- **ADL/mobility** → Section G
- **Diagnoses** → Section I
- **Medications** → Section M
